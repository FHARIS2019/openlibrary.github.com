trim(d)¶

Trims empty value from d.

>>> trim = edit().trim

>>> trim("hello ")
'hello'
>>> trim(['hello ', '', ' foo'])
['hello', 'foo']
>>> trim({'x': '', 'y': 'foo'})
{'y': 'foo'}
>>> trim({'x': '', 'unique': 'foo'})
>>> trim([{'x': '', 'y': 'foo'}, {'x': ''}])
[{'y': 'foo'}]
