_json¶

Wrapper to simplejson to fix unicode/utf-8 issues in python 2.4.

See Bug#231831 for details.

>>> loads(dumps(u'\u1234'))
u'\u1234'
>>> loads(dumps(u'\u1234'.encode('utf-8')))
u'\u1234'
>>> loads(dumps({'x': u'\u1234'.encode('utf-8')}))
{u'x': u'\u1234'}
