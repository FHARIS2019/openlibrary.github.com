readquery¶

class infogami.infobase.readquery.Query(conditions=None)¶

Query is a list of conditions. Each condition is a storage object with (“key”, “op”, “datatype”, “value”) as keys.

>>> q = Query()
>>> q
<query: []>
>>> q.add_condition("name", "=", "str", "foo")
>>> q
<query: ['name = str:foo']>
>>> q.add_condition('type', '=', 'ref', '/type/page')
>>> q.get_type()
'/type/page'
>>> q
<query: ['name = str:foo', 'type = ref:/type/page']>
