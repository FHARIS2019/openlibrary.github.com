lru¶

Infobase cache.

class infogami.infobase.lru.LRU(capacity, d=None)¶

Dictionary which discards least recently used items when size exceeds the specified capacity.

>>> d = LRU(3)
>>> d[1], d[2], d[3] = 1, 2, 3
>>> d[1], d[2], d[3]
(1, 2, 3)
>>> d[2] and d
[1, 3, 2]
>>> d[1] and d
[3, 2, 1]
>>> d[4] = 4
>>> d
[2, 1, 4]
>>> del d[1]
>>> d
[2, 4]
>>> d[2] = 2
>>> d
[4, 2]
