class infogami.infobase.client.Store(conn, sitename)¶

Store to store any arbitrary data.

This provides a dictionary like interface for storing documents. Each document can have an optional type (default is “”) and all the (type, name, value) triples are indexed.

clear()¶

Removes all keys from the store. Use this with caution!
