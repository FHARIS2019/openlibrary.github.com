infobase¶

Infobase: structured database.

Infobase is a structured database which contains multiple sites. Each site is an independent collection of objects.

class infogami.infobase.infobase.Infobase(store, secret_key)¶

Infobase contains multiple sites.

add_trigger(type, func)¶

Registers a trigger to call func when object of specified type is modified. If type=None is specified then the trigger is called for every modification. func is called with old object and new object as arguments. old object will be None if the object is newly created.
