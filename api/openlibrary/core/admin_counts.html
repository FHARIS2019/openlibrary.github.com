admin_counts¶

Script to read out data from thingdb and put it in couch so that it can be queried by the /admin pages on openlibrary

openlibrary.core.admin_counts.connect_to_pg(config_file)¶

Connects to the postgres database specified in the dictionary config. Needs a top level key db_parameters and under that database (or db) at the least. If user and host are provided, they’re used as well.
