init¶

Library for starting and monitoring Open Library services.

Many services need to be started to run an Open Library instance. It becomes quite complicated to manage them manually. This library provides init, a process manager like unix init, for starting and managing OL services.

This is used only for running the dev instance. In production, these services are typically run on multiple nodes and monitored using upstart.
