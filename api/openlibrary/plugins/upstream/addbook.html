openlibrary.plugins.upstream.addbook.trim_value(value)¶

Trim strings, lists and dictionaries to remove empty/None values.

>>> trim_value("hello ")
'hello'
>>> trim_value("")
>>> trim_value([1, 2, ""])
[1, 2]
>>> trim_value({'x': 'a', 'y': ''})
{'x': 'a'}
>>> trim_value({'x': [""]})
None
