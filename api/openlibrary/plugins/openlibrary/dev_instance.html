dev_instance¶

Special customizations for dev instance.

This module is imported only if dev_instance is set to True in openlibrary config.

class openlibrary.plugins.openlibrary.dev_instance.CoverstoreMiddleware(app)¶

Middleware to delegate all /cover/* requests to coverstore.

This avoids starting a new service for coverstore. Assumes that coverstore config is at conf/coverstore.yml
