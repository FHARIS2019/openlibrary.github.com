graphs¶

Utilities for rendering Graphite graphs.

class openlibrary.plugins.admin.graphs.GraphiteGraph¶

Representation of Graphite graph.

Usage:

g = GraphiteGraph() g.add(“stats.timers.ol.pageload.all.mean”).apply(“movingAverage”, 20).alias(“all”) print g.render()

In templates:

$ g = GraphiteGraph() $g.add(“stats.timers.ol.pageload.all.mean”).apply(“movingAverage”, 20).alias(“all”) $:g.render()
