solr¶

Python library for accessing Solr.

class openlibrary.utils.solr.Solr(base_url)¶
escape(query)¶

Escape special characters in the query string

>>> solr = Solr("")
>>> solr.escape("a[b]c")
'a\\[b\\]c'
select(query, fields=None, facets=None, rows=None, start=None, doc_wrapper=None, facet_wrapper=None, **kw)¶

Execute a solr query.

query can be a string or a dicitonary. If query is a dictionary, query is constucted by concatinating all the key-value pairs with AND condition.

openlibrary.utils.solr.urlencode(d, doseq=False)¶

There is a bug in urllib when used with unicode data.

>>> d = {"q": u"\u0C05"}
>>> urllib.urlencode(d)
'q=%E0%B0%85'
>>> urllib.urlencode(d, doseq=True)
'q=%3F'

This function encodes all the unicode strings in utf-8 before passing them to urllib.
